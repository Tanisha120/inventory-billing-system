## Inventory and Billing System

Inventory and Billing System is a JavaFX-based desktop application designed to manage inventory records and billing operations efficiently. The system allows users to add, update, delete, and search products while automatically tracking stock levels, pricing, and product expiry status. It provides an easy-to-use interface for inventory management and billing tasks.

## Features

### Inventory Management
- Add new products
- Update existing products
- Delete products
- Search products
- View complete inventory

### Billing System
- Generate bills
- Calculate total amount
- Update inventory after purchase

### Data Persistence
- Save inventory records to file
- Load inventory records automatically

### User Interface
- JavaFX-based graphical interface
- Interactive forms
- Table-based product display
- Easy navigation and management

## Technologies Used

### Programming Language
- Java

### Framework
- JavaFX

## Dataset / Storage

### Data Stored
The system stores information such as:
- Product ID
- Product Name
- Quantity
- Price
- Expiry Date
- Stock Status

### Storage Method
- Local file storage using inventory data files

## How to Run

### Step 1: Clone the Repository
git clone <repository-url>

### Step 2: Open the Project
Open the project in Eclipse, IntelliJ IDEA, or any Java IDE.

### Step 3: Configure JavaFX
Ensure JavaFX libraries are added to the project build path.

### Step 4: Run the Application
Run the MiniMartApp JavaFX application file.
